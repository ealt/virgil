# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |
| < 0.1.0 | No        |

## Reporting a Vulnerability

Please report security vulnerabilities via GitHub's private advisory form:
<https://github.com/ealt/virgil/security/advisories/new>

If you are unable to use the advisory form, open a GitHub issue with minimal details and request a private contact channel.
